# Termux Hacker Green-Black Theme

A dark, hacker-style green and black theme for Termux.

**Developed by حامد فریفته (Hamed Farifteh)**

## Features

- Classic hacker green text on black background
- Beautiful green prompt
- Matrix-style welcome banner
- Useful hacker-themed aliases
- Clean and professional look

## How to Use

### Step 1: Update Termux
```bash
pkg update && pkg upgrade -y
pkg install git figlet lolcat -y
```

### Step 2: Clone and run the script
```bash
git clone https://github.com/gemcancers-debug/termux-hacker-theme.git
cd termux-hacker-theme
chmod +x hacker_theme.sh
./hacker_theme.sh
```

### Step 3: Restart Termux
Close Termux completely and open it again to see the new hacker theme.

## What the Script Does

- Sets a green hacker-style prompt
- Changes terminal colors to green on black
- Adds a cool matrix-style banner on startup
- Adds useful aliases (`ll`, `matrix`, `hack`, etc.)

## Safety
This script only modifies your shell configuration (`.bashrc`). It does not harm your device or delete any files.
